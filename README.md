# 🎨 Matheus de Castro | Arts & Backend Dev Portfolio

Este é o repositório do meu portfólio pessoal, um espaço desenvolvido para unir o rigor lógico da engenharia de software com a sensibilidade estética da arte. Como estudante de TI na UFRN, utilizo este projeto para centralizar minhas soluções de arquitetura de backend, automações e pipelines inteligentes.

> "O código deve ser tão elegante quanto a arte."

## 🌐 Acesse o site

O portfólio está disponível em: **[matheusdecastro.com](https://matheusdecastro.com/)**

## 🚀 Tecnologias Utilizadas

Para garantir performance, componentização e um deploy sólido, o projeto utiliza:

* **React.js & Vite:** Estrutura modular, reativa e com build otimizado para produção.
* **JavaScript (ES6+):** Lógica de navegação baseada em estados e manipulação fluida de componentes.
* **CSS3 Moderno:** Estilização baseada em variáveis de escopo global, layouts flexíveis e responsividade nativa.
* **Google Fonts:** Tipografia focada em alta legibilidade.

## 🛠️ Funcionalidades e Diferenciais

* **Arquitetura Baseada em Componentes:** Divisão clara de responsabilidades (`HeadSite`, `Perfil`, `Skills`, `Experience`, `Projects`, `About`, `Hobbies`).
* **Navegação Inteligente por Estados:** Sistema de troca de telas fluido que permite transitar entre a Home e a seção de Hobbies/Artes sem recarregar a página.
* **Design Responsivo Avançado:** Interface totalmente adaptada para dispositivos móveis, tablets, desktops e telas ultrawide.
* **Favicon Indexado:** Identidade visual integrada com o robô de busca do Google através de metadados dinâmicos.
