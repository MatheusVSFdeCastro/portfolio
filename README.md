## 🚀 Tecnologias Utilizadas

Para este projeto, foquei em uma estrutura leve, performática e responsiva:

    HTML5 & CSS3: Estruturação semântica e estilização moderna com variáveis CSS e animações.

    JavaScript (Vanilla): Lógica para o efeito de typewriter e interatividade do usuário.

    DevIcons: Integração de badges tecnológicos para visualização rápida da stack.

    Google Fonts: Utilização da família Inter para máxima legibilidade.

## 🛠️ Funcionalidades e Diferenciais

    Efeito Typewriter: Uma introdução dinâmica que reflete minha identidade como "Arts & Dev".

    Design Responsivo: Adaptado para dispositivos móveis, tablets e desktops.

    Split-Card Interativo: Um elemento visual único que simboliza a dualidade entre Art e Coder.

    Seção de Skills Dinâmica: Visualização clara do meu progresso atual (incluindo o que estou estudando no momento, como React.js).

    Curadoria de Projetos: Vitrine dos meus melhores trabalhos em Python, C++, Java e SQL.
